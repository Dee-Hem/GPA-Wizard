# **App Name**: GPA Wizard

## Core Features:

- Course & Semester Entry: Users can add and organize course details (name, credits, grade) across different semesters for comprehensive academic tracking.
- Real-time CGPA Calculation: Instantly calculates and displays both the current semester GPA and the overall cumulative CGPA as data is entered or updated.
- Local Data Persistence: All entered course and semester data is saved directly in the user's browser storage for offline use and continuity between sessions.
- Course & Semester Management: Provides intuitive tools to edit or remove individual course entries or entire semesters, allowing for easy data correction.
- PDF Grade Report Export: Generate and download a printable PDF summary of all recorded academic data, including course breakdowns and calculated GPAs.
- Data Backup & Restore: Allows users to export their entire dataset to a local file for backup purposes and import it back into the calculator later.
- Academic Insight Assistant (AI Tool): An AI tool that, upon user request, provides generalized study tips or recommended learning strategies for common academic challenges, offering broad guidance without analyzing specific grades.

## Style Guidelines:

- Primary color: A confident and professional blue (#2E5FA6), selected to evoke reliability and academic progress, contrasting well with a light background.
- Background color: A very light, almost white-grey with a subtle blue tint (#F1F4F9), providing a clean and readable canvas consistent with a light color scheme.
- Accent color: A vibrant purple-blue (#8D6BF4) that is analogous to the primary color, chosen to highlight interactive elements and call-to-actions, ensuring good contrast and visual interest.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for conveying academic data clearly.
- Utilize clean, crisp outline or solid icons that clearly represent actions such as add, edit, delete, export, and import, maintaining a consistent and functional aesthetic.
- The layout will be clean and organized, with clear visual separation between semesters and course entries. Emphasis will be on responsive design to ensure optimal viewing and interaction across various device sizes.
- Subtle and smooth animations will be used for state changes, such as adding new courses or updating calculations, to provide visual feedback without distracting the user.
